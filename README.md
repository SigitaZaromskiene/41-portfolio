🏃 Run locally
Would like to run this project locally? Open terminal and follow these steps:

Clone the repo
git clone https://front-end-by-rimantas.github.io/40-grupe-portfolio-docland.git
Install NPM packages
npm i
or
npm install
Run the server
npm run dev
